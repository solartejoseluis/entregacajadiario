====== MODAL NUEVO DOMICILIO ======
Se usa para ingresar los nuevos domicilios ya sean internos o externos.

{{ :2023_07_11_modal_nuevo_domicilio.png?nolink&600 |}}

===== CARACTERÍSTICAS DEL MODAL =====

Este modal contiene algunos campos obligatorios, y una validación de llenado al momento de presionar el botón de confirmar agregar

===== BOTONES =====

- Botón Domi Interno
- Botón Domi Exerno
- Botón Confirma Agregar
- Botón Cancelar

===== CAMPOS: =====

  - Barrio:
  - # Factura
  - Valor Venta
  - H. Salida
  - Inyectología
  - Observaciones

===== DESCRIPCIÓN DE CADA ELEMENTO: =====

==== BOTONES ====

**Botón Domi Interno:** Muestra los campos específicos para domicilios Interno

**Botón Domi Externo** Muestra los campos específicos para domicilios Externo

**Botón Confirma Agregar** Inicia Validación de campos obligatorios mostrando mensajes en campos vacios, y cuando todo lo obligatorio esta lleno graba los datos en la tabla, cierra el modal y actualiza los listados de domicilios de la pantalla principal

**Botón Cancelar** Cierra el modal sin guardar datos.

==== CAMPOS ====

**Barrio:** Muestra en un listado (select) la lista de barrios

**# Factura:** Numero de factura, campo numérico para insertar el numero de factura


**Valor Venta** campo numérico para insertar el valor de venta.

**H. Salida** campo formato hora : HH:MM

**Inyectología** botón que cambia entre SI/NO

**Observaciones** campo de texto para agregar observaciones encontradas
